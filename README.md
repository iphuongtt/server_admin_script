
# server_admin_script
server administrator scripts
## Require
1. git
## Install
`cd /tmp`
`git clone https://github.com/iphuongtt/server_admin_script.git`
`cd server_admin_script`
1. For CentOS 7
	`sudo bash centos7/install.sh`
2. For Ubuntu 18
	`sudo bash ubuntu18/install.sh`
### Run
1. For root user
	`sudo server-admin`
2. For normal user
	`server-admin`
